# BlueBridgeEmbedded

蓝桥杯嵌入式方向练习与比赛工程集合，主要基于 STM32F10x 标准外设库和 Keil MDK 组织。仓库中既包含基础外设实验，也包含历届赛题与参考工程，适合作为复习、移植和功能对照的资料库。

## 仓库内容

- `01_LCD`：LCD 基础显示实验
- `02_LED`：LED 控制实验
- `03_KeyBoard`：按键输入实验
- `04_Buzzer`：蜂鸣器实验
- `05_TIM`：定时器基础实验
- `06_IIC&EEPROM`：I2C 与 EEPROM 实验
- `07_ADC`：ADC 采样实验
- `08_RTC`：实时时钟实验
- `09_USART`：串口通信实验
- `10_PWM`：PWM 输出实验
- `11_InputCaputer`：输入捕获实验
- `12_2016_第七届_蓝桥杯_省赛_电压测量监控设备`：2016 省赛相关工程
- `13_2020_第十一届_蓝桥杯_嵌入式设计与开发项目_省赛`：2020 省赛相关工程
- `14_PWMIC_EXA`、`15_IIC&LIS302DL`、`16_SEG_ADCKEY_DHT11_DS18B20`：综合外设练习
- `EXA参考程序`、`模板`、`底层`、`液晶驱动参考例程`：参考代码和移植基础

## 开发环境

- MCU/板卡：蓝桥杯嵌入式常见 STM32F10x 开发板
- IDE：Keil MDK
- 固件库：STM32F10x Standard Peripheral Library

## 使用方式

1. 进入对应实验目录。
2. 使用 Keil 打开目录中的 `.uvproj` 或 `.uvopt` 工程文件。
3. 根据自己的开发板连线情况检查时钟、引脚和外设配置。
4. 编译、下载并在板上验证实验现象。

## 说明

- 这是一个“多工程仓库”，目录之间相互独立，不是单一可直接编译的完整项目。
- 不同目录的代码风格和抽象层级并不完全一致，属于不同阶段的练习和沉淀。
- 若你只想快速上手，建议优先从 `01_LCD`、`02_LED`、`06_IIC&EEPROM`、`10_PWM` 开始。
